startAdaptiveV1(host, snippetID)

Start the Trusteer gathering process. Given a Trusteer JavaScript snippet ID, this function will inject that snippet into the webpage, initiating the gathering process. During gathering, two callback functions will be called; getTCSID, and getFlow. The injected Trusteer snippets will call getTCSID to indicate that we have been given a session ID. This session ID should be used for subsequent requests to the Adaptive Proxy SDK. The getFlow function will be called to indicate that the gathering process has completed.

Parameters:
Name Type Description
host string

The host of the Trusteer JavaScript snippet to inject into the webpage, e.g. a1bcdefgh2ijkl.cloudfront.net.
snippetID int | string

The identifier of the Trusteer JavaScript snippet to inject into the webpage, e.g. 123456.
